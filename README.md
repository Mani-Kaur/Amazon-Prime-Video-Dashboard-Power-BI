# Amazon Prime Video Dashboard – Power BI

##  Project Overview
This project analyzes Amazon Prime Video content using Power BI to identify trends in movies, TV shows, genres, ratings, countries, and release years. It transforms raw data into meaningful insights using interactive visualizations.

---

##  Dataset Features
- Movies and TV Shows data  
- Genre information  
- Ratings classification  
- Country-wise content distribution  
- Release year trends  

---

##  Tools Used
- Power BI  
- Power Query  
- DAX  
- Data Visualization  

---

## 📈 Dashboard Insights
- Drama is the most common genre  
- Movies dominate the platform  
- Most content is rated 13+  
- Content production increased rapidly after 2015  

---

##  Visualizations Included
- KPI Cards  
- Bar Charts  
- Donut Chart  
- World Map  
- Trend Analysis  

---

##  Learning Outcomes
- Data cleaning and transformation  
- Dashboard design principles  
- DAX calculations and measures  
- Interactive storytelling using data
